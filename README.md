# radio
Things and apps for radio development.

## Sorteador de nombres
En este repositorio se incluye una app lista para usar: `sorteador-nombres.html`.

### Uso rápido
1. Abre `sorteador-nombres.html` en tu navegador.
2. Escribe o pega un nombre por línea.
3. Pulsa **Sortear orden** para generar el orden aleatorio.

### Insertar en otra web
Puedes embeberla con un `iframe`:

```html
<iframe src="/sorteador-nombres.html" width="100%" height="650" style="border:0" loading="lazy"></iframe>
```
